Blood Bank Management System - Sample Project

Files:
- db_init.sql       : SQL to create database and tables. IMPORTANT: Replace the {PASSWORD_PLACEHOLDER} token with a bcrypt hash of a password before running, or remove the INSERT line.
- config.php        : Database connection and session helpers.
- index.php         : Home page.
- register.php      : Sign up page.
- login.php         : Login page.
- logout.php        : Logout.
- donor.php         : Donor add/list page.
- receiver.php      : Receiver add/list page.
- admin.php         : Admin user list.
- styles.css        : Simple styling.
- script.js         : Placeholder JS.

Setup:
1. Install PHP, MySQL (or MariaDB), and a web server (Apache/Nginx).
2. Place files in your webroot (e.g., /var/www/html/blood_bank_sample).
3. Create the database:
   - Option A (recommended): Edit db_init.sql: replace {PASSWORD_PLACEHOLDER} with a hash generated by PHP:
       <?php echo password_hash('admin123', PASSWORD_DEFAULT);
     then run the SQL.
   - Option B: Remove the INSERT line and create an admin account via the registration form then manually change role to 'admin' in DB.
4. Update DB credentials in config.php.
5. Visit the site in your browser.

Security notes:
- This is a simple demo. For production, use prepared statements (already used), CSRF protection, input validation, HTTPS, and secure password policies.
